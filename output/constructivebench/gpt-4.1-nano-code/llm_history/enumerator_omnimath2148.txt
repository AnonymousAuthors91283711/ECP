\[

Let's analyze the expression \( k = \sqrt{100 + \sqrt{n}} + \sqrt{100 - \sqrt{n}} \):

- Set \( x = \sqrt{n} \).
- Then \( k = \sqrt{100 + x} + \sqrt{100 - x} \).

Squaring both sides:

\[
k^2 = (\sqrt{100 + x} + \sqrt{100 - x})^2 = 100 + x + 100 - x + 2\sqrt{(100 + x)(100 - x)} = 200 + 2\sqrt{100^2 - x^2}

Rearranged,
