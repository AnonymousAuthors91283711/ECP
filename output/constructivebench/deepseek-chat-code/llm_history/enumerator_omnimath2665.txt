The problem is to find the value of the expression \((-1)^3 + (-1)^2 + (-1)\). We can compute this directly by evaluating each term and summing them up.

```python
# Compute (-1)^3 + (-1)^2 + (-1)
term1 = (-1) ** 3
term2 = (-1) ** 2
term3 = -1
result = term1 + term2 + term3
print(result)
```
